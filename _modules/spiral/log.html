        """The logging class provided to all user functions.

        When testing the model locally, calls to `spiral.log` will simply print
        the input `logs` dictionary to stdout.

        Args:
            logs:
                A dictionary that contains key-value pairs to be logged. The
                keys are of type `string` and the values should be JSON
                serializable. To log non-JSON serializable values such as numpy
                arrays and binary data, encode the values using `spiral.encodeNumpy`
                and `spiral.encodeBinary` before passing them to the logger. If
                no explicit encoding is done for non-JSON serializable values,
                we will attempt to infer the value types and perform the
                appropriate encoding.
        """
        print(logs)
