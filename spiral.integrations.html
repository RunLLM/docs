config(filepath: str, save_mode: spiral.enums.GoogleSheetsSaveMode = GoogleSheetsSaveMode.OVERWRITE) → spiral.operators.SaveConfig[source]

Configuration for saving to Google Sheets Integration.

Parameters

  • filepath – Google Sheets filepath to save to. Any directory in the path that does not exist will be created.

  • save_mode –

    The save mode to use when saving this artifact to Google Sheets. Possible values are: - OVERWRITE: If a spreadsheet with the same name exists, it will overwrite it.

    Otherwise, it will create a new spreadsheet.

    • NEWSHEET: If a spreadsheet with the same name exists, it will create a new sheet for this artifact.

      Otherwise, it will create a new spreadsheet.

    • CREATE: This will create a new spreadsheet, even if one with the same name exists. The previous

      spreadsheet will not be deleted, as Google Sheets allows for multiple spreadsheets of the same name.

Returns

SaveConfig object to use in TableArtifact.save()
