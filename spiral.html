  • entry_point – The relative path, to the directory specified by path, of the function’s entry point. Default to model.py if not specified.

  • class_name – The class name of the operator. If not specified, it will default to Function. If specified as empty string, the method will be directly imported from the entry_point_file.

  • method –

    The method to run on artifacts as the operator. If class_name is specified, we first construct the class and call .<method>(inputs) of the object. If class_name is empty string, we import <method> from <entry_point>, and call the method over inputs directly.

    Default to predict if not specified.

    See Example below for more details.

  • op_name – The operator name in the repo’s .aqconfig field. This will override all other arguments. Follow examples/.aqconfig.example for more details.

A metric artifact can only be created from applying a metric operator to a table artifact. For example:

class spiral.table_artifact.TableArtifact(api_client: spiral.api_client.APIClient, dag: spiral.dag.DAG, artifact_id: uuid.UUID)[source]

Bases: spiral.generic_artifact.Artifact

This class represents a computed table within the flow’s DAG.

An artifact can only be created from a Flow object, or by applying operators to an existing artifact. For example:

client = spiral.SpiralClient(…) flow = client.create_flow(“foo”) tableArtifactFromFlow = flow.sql(“SELECT * …”) tableArtifactFromOperator = transform(‘function.zip’, tableArtifactFromFlow)

Alternatively:

@op def foo(df):

# the contents of in function.zip here return result

tableArtifactFromOperator = foo(tableArtifactFromFlow)

The contents of these artifacts can be manifested locally or written to an integration. For example:

data = tableArtifact.to_df() tableArtifact.save(“integration/output_table”)

The contents of these artifacts can also be tested. For example:

passed, logs = table_artifact.add_unittest(

name=”foo”, description=”This is an illustrative example”, file_path=”unittest.zip”,

)

Alternatively:

@add_unittest def foo_test(df: pd.DataFrame):

return df[“year”] < 2022

passed, logs = foo_test(table_artifact)
